JavaScript Loops Palette

1. The For Loop

Used when we know exactly how many times to repeat.

Increment (Adding 1) 
for (let i = 1; i <= 3; i++) {
  console.log(i);
}
Result:
1, 2, 3
Decrement (Subtracting 1) 
for (let i = 3; i >= 1; i--) {
  console.log(i);
}
Result:
3, 2, 1
2. The While Loop

The "Watchman": Runs as long as the condition stays true.

Increment (Adding 2) 
let i = 0;
while (i <= 4) {
  console.log(i);
  i += 2;
}
Result:
0, 2, 4
Decrement (Subtracting 2) 
let i = 10;
while (i >= 6) {
  console.log(i);
  i -= 2;
}
Result:
10, 8, 6
3. The Do...While Loop

The "First Try": Always runs at least once before checking.

Increment (At least once) 
let i = 0;
do {
  console.log(i);
  i++;
} while (i < 1);
Result:
0
Decrement (Countdown) 
let i = 2;
do {
  console.log(i);
  i--;
} while (i > 0);
Result:
2, 1
