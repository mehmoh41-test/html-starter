JavaScript Loops Reference

1. The FOR Loop

Ideal for counting. All control logic is in one line.

Increment (Counting Up) 
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
Result:
1, 2, 3, 4, 5
Decrement (Counting Down) 
for (let i = 5; i >= 1; i--) {
  console.log(i);
}
Result:
5, 4, 3, 2, 1
2. The WHILE Loop

Checks the condition BEFORE running the code.

Increment (Up by 10s) 
let i = 0;
while (i <= 30) {
  console.log(i);
  i += 10;
}
Result:
0, 10, 20, 30
Decrement (Down by 5s) 
let i = 15;
while (i >= 0) {
  console.log(i);
  i -= 5;
}
Result:
15, 10, 5, 0
3. The DO...WHILE Loop

The "Do it anyway" loop. Checks the condition AFTER running once.

Increment (Guaranteed Execution) 
let i = 100;
do {
  console.log(i);
  i++;
} while (i < 10);
Result:
100 (Runs once even though 100 is not less than 10!)

💡 Important Notes to Remember

  • Initialization: Where the loop starts (e.g., let i = 0).
  • Condition: The loop keeps running while this is true (e.g., i < 10).
  • The Change: If you forget to increment (i++) or decrement (i--), you create an Infinite Loop which can crash your browser!
  • Incrementing: Usually moves from a lower number to a higher number using <.
  • Decrementing: Moves from a higher number to a lower number using >.
  • When to use what? Use for for known counts, while for unknown durations, and do...while when you need at least one result.
